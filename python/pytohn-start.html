  • 'MyClass' একটি ক্লাস ও 'myObj' ঐ ক্লাসের অবজেক্ট। এবং 'myObj.num' দ্বারা ক্লাসের প্রপার্টি কে এবং 'myObj.myFun()' দ্বারা ক্লাসের মেথড কে কল করা হয়েছে।

    
class MyClass:
  num = 5 #ক্লাস বডি
  def myFun(self): # ঐচ্ছিক ফাংশান #ক্লাস বডি
    print('hello form function') #ফাংশান বডি
myObj = MyClass() #ক্লাসের অবজেক্ট/ইনস্ট্যান্স  তৈরি
print(myObj.num) #প্রপার্টি কল
myObj.myFun() #মেথড কল
