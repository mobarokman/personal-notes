রিয়েক্ট জেএস কি? Last updated: 2019-06-01

ReactJS ইউজার ইন্টারফেস তৈরির জন্য একটি জাভাস্ক্রিপ্ট লাইব্রেরি

১.১ শুরুর আগে

  • react কে ইম্পোর্ট(ES6 কনসেপ্ট) করতে: import React from 'react'
  • ReactDOM কে ইম্পোর্ট করতে: import ReactDOM from 'react-dom'
  • ReactDOM.render এই মেথডটি ২টি আরগুমেন্ট নেয় রেন্ডার ইলিমেন্ট এবং রেন্ডার স্কোপ: ReactDOM.render(<h1>Hello, world!</h1>,document.getElementById('root'))

১.২ কম্পোনেন্ট

কম্পোনেন্ট দুই প্রকার। ফাংশানাল কম্পোনেন্ট ও ক্লাস কম্পোনেন্ট।

ফাংশানাল কম্পোনেন্ট :

function Welcome(props) {
	return <h1>Hello, {props.name}</h1>;
}
ক্লাস কম্পোনেন্ট :

function Welcome(props) {
	return <h1>Hello, {props.name}</h1>;
}
