CSS in ReactJS

  • Inline CSS: property name will be camelCase, and values are string. like- backgroundColor: "red" 

    
<h1 style={{color: "red", backgroundColor: "blue"}}>Hello, World</h1>

  • Object CSS: make a object and call it in style attribute

    
class Show extends Component {
	render(){
		const myStyle = {
			color: "Red",
			backgroundColor: "lightBlue",
			padding: "10px",
			fontFamily: "Arial"
		}
		return (
			<h1 style={myStyle}>Hello, World</h1>	
		)
	}
}

export default Show;

  • Normal External CSS: make a .css file, import it and call classname in jsx 

    
import '../css/mystyle.css'
<h1 className="foo boo">Hello, World</h1>

  • CSS Module: make a .module.css file, import it and call under classname as js object jsx 

    
import style from './mystyle.module.css'
<h1  className={style.boo}>Hello, World</h1>
