১.২ ভেরিয়েবল ও ডাটা টাইপ

ভেরিয়েবল ডিক্লেয়ারেশন/তৈরি: var a = "হ্যালো ওয়ার্ল্ড";

ভ্যারিয়েবলের নাম দেওয়ার ক্ষেত্রে সাধারণ নিয়মগুলো হচ্ছে:
  • নামের মধ্যে অক্ষর(a, x, z), ডিজিট(1, 2, 3), আন্ডারস্কোর(_) এবং ডলার($) চিহ্ন থাকতে পারে
  • নাম অবশ্যই অক্ষর দিয়ে অথবা $ এবং _ দিয়েও নাম শুরু হতে পারে
  • নাম কেসসেন্সিটিভ (যেমন a এবং A দুটি আলাদা ভ্যারিয়েবল)
  • সংরক্ষিত শব্দ/reserved keyword (যেমন- জাভাস্ক্রিপ্ট কিওয়ার্ড) নাম হিসাবে ব্যবহার করা যায়না

  • camelCase জাভাস্ক্রিপ্ট এ ক্যামেলকেস ইউজ করা একটা কনভেনশন/ মানসম্মত:

    camelCase myLIfe sakibKhan

ডাটা টাইপ:

জাভাস্ক্রিপ্ট এ ব্যাসিকিলি দুই ধরনের ডাটা টাইপ আছেঃ

  • প্রিমিটিভ ডাটা টাইপ: প্রিমিটিভ টাইপের ডাটাগুলো তে ভ্যালু সরাসরি স্টোর করা থাকে। আমরা জানি জাভাস্ক্রিপ্ট এ সবই অবজেক্ট। কিন্তু এই প্রিমিটিভ ডাটা টাইপগুলো ছাড়া। এগুলা অবজেক্ট না। যেমনঃ (১) স্ট্রিং (২) নাম্বার (৩) বুলিয়ান
  • নন প্রিমিটিভ/রেফারেন্স ডাটা টাইপ: নন-প্রিমিটিভ ডাটা টাইপের ভ্যালু সরাসরি সেইভ করা থাকে না। বরং ভ্যালুর রেফারেন্স সেইভ থাকে। আর এই টাইপের ডাটা অবজেক্ট। মানে এদেরও আবার অনেক প্রোপ্রার্টি আছে। যেমনঃ (১) অ্যারে (২) অবজেক্ট (২) ফাংশন

কিছু ডাটা টাইপঃ স্ট্রিং(String), সংখ্যা(Number), বুলিয়ান(Boolean), অ্যারে(Array), অবজেক্ট(Object)।


var name = "Tamim";                 // String
var scores = 16;                    // Number
var isOkk = true;					// Boolean
var animals = ["Tiger", "Lion", "Elephant"];    // Array
var a = {firstName:"Azizur", lastName:"Rahman"};// Object
ডাটা টাইপ নিয়ে কিছু কথাঃ

  • var a = 28 + "Tamim"; //28Tamim এবং var a = 28 + "Tamim"; //28Tamim উভয়ই সমান। string এর সাথে number যোগ করলে তা string হিসাবে গণ্য হয়।

  • জাভাস্ক্রিপ্ট এক্সপ্রেশনকে বাম থেকে ডান দিকে সম্পাদন করে। ডাটা টাইপের ধারাবাহিকতা ভিন্ন ভিন্ন ফলাফল প্রদর্শন করতে পারেঃ var a = 30 + 10 + "Tamim"; //40Tamim এবং var a = "Tamim" + 28 + 12; //Tamim2812

  • typeof অপারেটর(operator) ব্যবহার ভ্যারিয়েবলের টাইপ জানতে পারি:

    typeof "Tamim" //string

  • কনকাটিনেশন একটা স্ট্রিং/নাম্বার অন্য একটা ট্রিং এর সাথে একসাথে করার জন্যঃ কনকাটিনেশন(+) যোগ চিহ্ন ব্যবহার করা হয়ঃ

    var a = "Tamim" +"Sakib" + 28;

1.3 অপারেটর

২ + ৪ এ ৬ হয়, এখন এখানে + হচ্ছে অপারেটর, আর ২ এবং ৪ হচ্ছে অপারেন্ড

জাভাস্ক্রিপ্ট অপারেটর সমুহু:

গাণিতিক/অ্যারিথমেটিক অপারেটর:

অপারেটর বর্ণনা
+ যোগ(Addition)
- বিয়োগ(Subtraction)
* গুণ(Multiplication)
/ ভাগ(Division)
% (ভাগশেষ)Modulus
++ এক করে বৃদ্ধি (Increment)
-- এক করে হ্রাস(Decrement)

এসাইনমেন্ট অপারেটর:এসাইনমেন্ট অপারেটর জাভাস্ক্রিপ্ট ভ্যারিয়েবলের মধ্যে ভ্যালু এসাইন করে।

অপারেটর উদাহরণ বর্ণনা
= a=b
+= a+=b a= a+b
-= a-=b a= a-b
*= a*=b a= a*b
/= a/=b a a/b
%= a%=b a= a%b
